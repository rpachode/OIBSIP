# BMI Calculator

## Project Description
BMI Calculator is a Python-based application that calculates the Body Mass Index (BMI) of a person using their weight and height. It provides the calculated BMI value based on user input.

## Technologies Used
- Python

## Features
- Takes user weight and height as input
- Calculates BMI accurately
- Displays the BMI result
- Simple and beginner-friendly interface

## How to Run the Project

1. Install Python on your system.
2. Download or clone this repository.
3. Run the Python file:
   python BMI_Calculator.py
4. Enter your weight and height when prompted.

## Example Output
Enter weight in kg: 55
Enter height in meters: 1.65

Your BMI is: 20.20
